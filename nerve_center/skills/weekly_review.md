# Skill: Weekly Review & CEO Briefing

## Description
Generate comprehensive weekly review and CEO briefing document every Friday.

## Trigger
- Scheduled: Every Friday at 4:00 PM
- Manual: "generate weekly review"

## Inputs
- Activity logs from the week
- Dashboard data
- Business_Goals.md
- Financial records

## Process

### Step 1: Gather Data
```
Collect from logs:
- All processed events
- Emails sent/received
- Tasks completed
- Invoices processed
- Meetings scheduled
- Errors encountered
```

### Step 2: Calculate Metrics
```
Weekly KPIs:
- Tasks completed vs created
- Response time average
- Revenue received
- Expenses incurred
- Client interactions
- System uptime
```

### Step 3: Compare to Goals
```
From Business_Goals.md:
- Weekly targets
- Monthly progress
- Quarterly objectives
- Flag any at-risk goals
```

### Step 4: Identify Highlights
```
Find:
- Biggest wins
- Notable achievements
- Important milestones reached
```

### Step 5: Surface Issues
```
Flag:
- Overdue items
- Unresolved errors
- Pending approvals
- Bottlenecks
```

### Step 6: Generate Recommendations
```
AI suggestions for:
- Process improvements
- Priority adjustments
- Resource allocation
- Risk mitigation
```

### Step 7: Create Briefing Document

```markdown
# CEO Briefing - Week of {{date}}

## Executive Summary
[2-3 sentence overview of the week]

## Key Metrics
| Metric | This Week | Last Week | Change |
|--------|-----------|-----------|--------|
| Tasks Completed | X | Y | +/-Z% |
| Emails Processed | X | Y | +/-Z% |
| Revenue | $X | $Y | +/-Z% |
| Expenses | $X | $Y | +/-Z% |

## Wins This Week
- Win 1
- Win 2
- Win 3

## Challenges & Blockers
- Challenge 1 - [Status/Resolution]
- Challenge 2 - [Status/Resolution]

## Financial Summary
- Total Received: $X
- Total Spent: $X
- Outstanding Invoices: $X
- Pending Approvals: X items

## Goal Progress
[Progress bars or status for each quarterly objective]

## Next Week Focus
1. Priority 1
2. Priority 2
3. Priority 3

## Items Requiring Your Attention
- [ ] Approval needed: [Item]
- [ ] Decision needed: [Item]
- [ ] Review needed: [Item]

## AI Recommendations
- [Recommendation 1]
- [Recommendation 2]

---
*Generated by AI Employee on {{timestamp}}*
```

## Output
- Save to `nerve_center/briefings/week_{{date}}.md`
- Update Dashboard with briefing link
- Notify owner that briefing is ready

## Follow-up Actions
- If critical issues found: Notify immediately, don't wait for Friday
- If goals at risk: Create priority tasks for next week
- If errors recurring: Suggest system improvements
